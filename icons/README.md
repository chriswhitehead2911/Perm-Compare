# Icons

Place the following PNG files in this directory before submitting to the Chrome Web Store:

| File | Size | Used for |
|---|---|---|
| `icon16.png` | 16×16 | Toolbar button, browser favicon |
| `icon32.png` | 32×32 | Windows taskbar |
| `icon48.png` | 48×48 | Extensions management page (chrome://extensions) |
| `icon128.png` | 128×128 | Chrome Web Store listing tile |

## Design notes

- Use a transparent background
- The icon should be recognisable at 16×16 — keep it simple
- Suggested concept: two overlapping shield or lock shapes (representing permission comparison), or the letters "PC" in a Salesforce-blue (#0176D3) rounded square
